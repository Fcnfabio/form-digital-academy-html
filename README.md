# DIGITAL ACADEMY

- Formulario de inscripción moderno y responsive para Digital Academy, plataforma de cursos online. Desarrollado con HTML5 semántico y CSS3 con variables personalizadas.

# Descripción

- Formulario de inscripción para Digital Academy, una academia de cursos online especializados en tecnología. Diseñado con enfoque en:
* Validación nativa de HTML5
* Diseño responsive y centrado
* Accesibilidad con etiquetas semánticas
* Paleta de colores violeta moderna y profesional

#  Características funcionales

- Campo de nombre completo con autocompletado
- Campo de correo electrónico con validación de formato
- Campo de edad con input numérico
- Selector de categoría de curso (UX/UI, Marketing, Data Science, Ciberseguridad)
- Selección de modalidad con radio buttons (En línea, Presencial, Híbrido)
- Selección de nivel de dificultad (Principiante, Intermedio, Avanzado)
- Checkbox de política de privacidad y reembolso
- Botones de enviar y limpiar formulario
